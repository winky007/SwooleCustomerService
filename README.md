# Swoole Customer Service DEMO

    使用Swoole扩展，PHP开发的一个客服聊天demo，类似facebook网页聊天界面

## 后台

    Swoole
    Redis
    Mysql

## 前台

    Jquery
    Jquery.ui.chatbox

## 配置

    global.php
    
## 地址

    后台URL: /admin/index.php
    前台URL: /index.php
    Swoole后端服务文件: websocket_server.php
